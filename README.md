# claude-plugins

A small personal marketplace of [Claude Code](https://claude.com/claude-code) plugins, by [Charbel Wakim](https://charbelwakim.com).

## Install

```text
/plugin marketplace add cwakim/claude-plugins
/plugin install <plugin-name>
```
